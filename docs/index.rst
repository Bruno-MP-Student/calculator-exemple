.. Welcome to calculator's exemple documentation!
=============================================

This is a simple project to demonstrate Python project organization with GitHub, pytest, and Sphinx.

Features
--------
- Add two numbers
- Subtract two numbers
- Automatically generate documentation with Sphinx
- Run tests with pytest

Usage
-----
Here's an example of how to use this project:

.. code-block:: python

    from calculator import add, subtract

    print(add(3, 5))  # Output: 8
    print(subtract(10, 4))  # Output: 6
